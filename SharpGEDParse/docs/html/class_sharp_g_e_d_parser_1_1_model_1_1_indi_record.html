◆ FullName

string SharpGEDParser.Model.IndiRecord.FullName
get

The individual's full name, using the first (preferred) name record.

Will be an empty string if the person has no name data.

Example: The GEDCOM record of "1 NAME John Doe /Jones/ Jr." will be returned by this property as "John Doe Jones Jr.".
