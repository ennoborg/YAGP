◆ ChildCount

int SharpGEDParser.Model.FamRecord.ChildCount
getset

The value of the NCHI tag from the GEDCOM.

Will be -1 if the NCHI tag was not specified. This field value is not guaranteed to match the number of entries in the Childs list.

If this value is -1 the writer will not emit the NCHI tag.
