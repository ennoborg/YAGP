Detailed Description

Contact information data.

The ADDR and associated CONT tag(s) should be used to contain the full "mailing label" address. All other address fields are 'optional' and recorded for backward compatibility.

The standard limits PHON, EMAIL, FAX and WWW entries to three, but that limit is not enforced.
